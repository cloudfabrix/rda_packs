## Overview

The **Config Backup** pack adds configuration backup and syslog-driven config management for network devices. It provides dashboards to view running configuration, track backup jobs, compare configuration changes. 



## Quick Start

| Step | Description |  
|------|-------------|  
| 1    | Activate the solution pack. After the Pack is in `ACTIVATED` status, use `Enable Single Tenant` menu option to enable the pack.  |  
| 2    | Use `Launch Dashboard` menu option to navigate to the pack's dashboard. Note: For multinent tenant environment, the pack specific dashboards will be available in Customer Ops page.|    
| 3   | Use `Config Management` ->  `Config Backup`.In Config Backups report click on `Backup All` to run Backup for all network devices. | 
| 4   | Use `Config Management` ->  `Config Backup`.In Config Backups report click on row level action `Backup Now` to get config Backup for a device. | 
| 5   | Use `Config Management` ->  `Config Backup`.In Config Backups report click on row level action `Config History` to view all the config for a device. | 
| 6   | In `Config History` click on row level actions `View Config` to view complete configuration , `View Difference` to get the comparison between the configurations , `Mark Golden Config` -> `Add` to mark the configuration as golden| 
| 7   | Use `Config Management` ->  `Config Backup Jobs` to view the jobs.|
